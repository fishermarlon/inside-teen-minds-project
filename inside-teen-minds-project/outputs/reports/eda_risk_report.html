df["risk_score"] = 0

# +2 points for high stress
df.loc[df["stress_level"] == "High", "risk_score"] += 2

# +1 point for low mood
df.loc[df["mood"] <= 3, "risk_score"] += 1

# +1 point for sleep less than 6 hours
df.loc[df["sleep_hours"] < 6, "risk_score"] += 1

# +1 point for low support (if scale is 15)
df.loc[df["support_feeling"] <= 2, "risk_score"] += 1

# Create a new column called 'risk_group'
def classify_risk(score):
    if score >= 4:
        return "high_risk"
    elif score >= 2:
        return "moderate"
    else:
        return "low_risk"

df["risk_group"] = df["risk_score"].apply(classify_risk)

Step 4 Summary: Risk Segmentation

Using a custom risk scoring model based on stress level, mood, sleep hours, and support feeling, each student was classified into a risk group:

  • High-Risk Students: Scored 4 or more showing multiple signs of emotional strain and reduced support
  • Moderate Risk: Scored 23 may benefit from proactive support or monitoring
  • Low Risk: Scored 01 currently stable across key wellness metrics

This segmentation allows schools or mental health programs to quickly identify which students may need immediate outreach and which groups can be used to study resilience factors.
